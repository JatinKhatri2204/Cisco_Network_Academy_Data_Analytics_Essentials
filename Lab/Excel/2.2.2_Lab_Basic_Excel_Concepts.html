Part 1: Basic Excel Formulas

There are two basic ways to perform calculations in Excel: formulas and functions. Formulas are expressions that operate on the values in a cell or range of cells; functions are predefined formulas built into Excel.

Step 1: Open a Blank Workbook

  1. If using the free version of Excel, log into Office.com, click on Excel and then click on New blank workbook.

Step 2: Typing a Formula Inside a Cell.

Excel formulas begin with an equal “=” sign and are typed directly into a cell or into the Formula Bar, which is above the columns.

  1. Click on cell A1 to make it the active cell.

  2. Type the following formula: = 1+1 and then Enter on the keyboard. This formula adds 1 and 1 together. Notice that cell A1 shows the result of the formula, 2, and the formula itself is shown in the formula bar when the cell is active.

  1. Select cell A2 and enter the formula = 4-2. This formula subtracts 2 from 4.

  2. Select cell A3 and enter the formula = 2*4. This formula multiplies 2 by 4.

  3. Select cell A4 and click in the formula bar. In the formula bar enter the formula = 6/2. This formula divides 6 by 2 and puts the results in cell A4.

Part 2: Basic Excel Functions

Excel has many built-in functions, or predefined formulas, that are used to perform both simple and complex calculations. Five basic, built-in functions we will cover in this lab are Sum, Average, Count, Max, and Min.

Step 1: Add a New Worksheet

  1. Click the plus “+” button at the bottom left of the workbook screen to add a new worksheet.

  2. Right click the new Sheet2 and rename it “Functions.”

Step 2: Explore the Excel Function Library

  1. Click on the Formulas tab in the toolbar to view the types of built-in functions in the Function Library group.
    You may have to expand the toolbar by clicking the down arrow at the far right of the tool bar to see the Function Library group.
  2. Click the dropdown arrow under the AutoSum Function. Note the five basic functions in this subgroup are as follows:
    1. Sum: adds the numeric values in the referenced cells
    2. Average: averages the numeric values in the referenced cells
    3. Count Numbers: counts how many referenced cells there are
    4. Max: returns the highest numeric value in the set of referenced cells
    5. Min: returns the lowest numeric value in the set of referenced cells

Step 3: Using the SUM function

  1. Click on Cell A1 and enter the value 10.
  2. In Cells A2 through A5, enter the following values:
    Cell A2 = 7
    Cell A3 = 5
    Cell A4 = 9
    Cell A5 = 12
  3. Click on Cell A6 to make it the active cell.
  4. With Cell A6 selected, click the down arrow under the AutoSum button in the Function Library and then select Sum.

Notice that once you click Sum with this cell selected, Excel automatically assumes you want to add together all of the values in the cells above cell A6 and puts a blinking dashed boarder around those cells.

It also puts the matching formula in cell A6 of =SUM(A1:A5). The cell range A1:A5 in the formula is highlighted to show the cells that Excel is assuming you want to add together.

Note that Excel uses a colon “:” to indicate that all the cells from A1 through A5 are to be added together. This is a more efficient way to write the formula than =A1+A2+A3+A4+A5.

  1. Press Enter on the keyboard to accept the formula. The value of 43 should now be in cell A6.
  2. Insert a new Row 1, and in the new cell A1, enter the text “SUM” as shown below.

Step 4: Using the Average Function

  1. In cell B1, enter the text “AVERAGE”.

  2. In cells B2 through B6 put in the values 10, 7, 5, 9, and 12, so they match the values in cells A2 through A6.

  3. Select the blank cell B7 and apply the Average function by clicking the down arrow under the AutoSum button in the Function Library and then selecting Average.

    Again, once you elect Average, Excel automatically assumes you want to average all of the values in the cells above cell B7 and puts a blinking dashed boarder around those cells. It also puts the appropriate formula in cell B7 of =Average(B2:B6).

  1. Press Enter on the keyboard. The value 8.6, which is the average of 10, 7, 5, 9, and 12, should now be in cell B7.

Step 5: Using the COUNT Function

  1. In cell C1, enter the text “COUNT”.

  2. Fill in the values for cells C2 through C6 by copying cells B2 through B6 and pasting them into cells C2 through C6.

    1. To copy the cells, select cell B2 and drag the mouse to select the range of cells B2 through B6. Using the keyboard, press CTRL+C to copy.

    2. To paste the contents into cells C2 through C6, select cells C2 through C6 and use the keyboard press CTRL+V to paste the content.

  3. Select the blank cell C7 and apply the Count function by clicking the down arrow under the AutoSum button in the Function Library and then selecting Count.

    The Count Numbers function =COUNT(C2:C6) will count the number of cells automatically included in the formula, which is 5.

Step 6: Using the MIN and MAX Functions

  1. Repeat the processes in steps 4 and 5 above for the MIN and MAX functions.

  2. Copy and paste the values of 10, 7, 5, 9, and 12 the cell ranges D2 through D6 and E2 through E6.

  3. In cell D7, apply the MIN function

    The Min function =MIN(D2:D6) returns the lowest of the values in the cells, which is 5.

  4. In cell E7, apply the MAX function

    The Max function =MAX(E2:E6) returns the highest of the values in the cells, which is 12.

If the MIN function finds the smallest value and the MAX function finds the largest value, what function is used to find the middle value? How would the formula be written for calculating the middle value in cells F2 through F6?
